# MyCn18: IDE para Desarrollo y Ejecución de Código con IA

**MyCn18** es un entorno de desarrollo (IDE) diseñado para construir, auditar y ejecutar lógica de programación apoyada por inteligencia artificial. A diferencia de otros editores, MyCn18 permite visualizar el proceso de pensamiento de la IA en tiempo real con **thinking graph**, gestionar tareas mediante un **tablero Kanban automatizado**, simular resultados de decisiones antes de aplicarlas con un **Ghost Runner** y revisar los cambios mediante un diagrama del código base.

#### Créditos e Inspiración

Quisiera dar el crédito intelectual de la visión de un "IDE de auditoría de agentes" a **[Toni Soriano](https://www.linkedin.com/in/toni-soriano-76218b32/?originalSubdomain=es)**. Toni tuvo la idea magistral de replantear cómo interactuamos con la IA en el código antes que yo. Este proyecto es mi interpretación técnica y personal de ese concepto, buscando llevarlo a la práctica con un enfoque en simulación y transparencia total.

#### Estado actual

MyCn18 es un proyecto personal en fase inicial de desarrollo. El objetivo es crear una herramienta que yo mismo, como desarrollador que prefiere el código a las interfaces cerradas, quiera usar todos los días.
